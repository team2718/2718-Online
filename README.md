# 2718-online

2718 Online is an FRC scouting web app for FRC team 2718 (and friends). 2718 Online will feature:

- Recording scouting reports for individual robots during a match
- Recording pit scouting
- Viewing combined data from scouting, TBA, Statbotics, etc.
- Creating pick lists

## Developing

Clone this repo and install dependencies with

```sh
pnpm install
```

Start a development server

```sh
pnpm run dev
```

## Building

To create a production version:

```sh
pnpm run build
```

```sh
pnpm run preview
```
